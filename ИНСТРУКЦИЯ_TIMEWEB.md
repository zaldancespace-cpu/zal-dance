## Шаг 1: Загрузка бэкенда (API)

### 1.1. Войди в Timeweb Cloud
Перейди на https://timeweb.cloud и войди в аккаунт.

### 1.2. Создай приложение
1. В левом меню нажми **"Apps"** (или "Приложения")
2. Нажми кнопку **"Создать"**
3. Выбери **"Node.js"**
4. Настройки:
   - **Название**: `zal-dance-api`
   - **Версия Node.js**: `18` или `20`
   - **Тариф**: выбери минимальный (от 0₽ или ~150₽/мес)
5. Нажми **"Создать"**

### 1.3. Загрузи код бэкенда
1. В приложении перейди во вкладку **"Код"** или **"Файлы"**
2. Загрузи файлы из папки `backend-timeweb/`:
   - `index.js`
   - `package.json`
3. Или используй Git: 
   - Создай репозиторий на GitHub
   - Подключи его к Timeweb

### 1.4. Настрой переменные окружения
1. Перейди во вкладку **"Переменные"** или **"Environment"**
2. Добавь переменные:
   ```
   YOOKASSA_SHOP_ID = 1216529
   YOOKASSA_SECRET_KEY = live_4hRLpFZQo6DbqC7AsQELLmdEV57lHW7s2-vj2ifUlFQ
   FRONTEND_URL = https://твой-сайт.timeweb.cloud
   ```
   ⚠️ FRONTEND_URL обновишь после создания фронтенда!

### 1.5. Запусти приложение
1. Нажми **"Деплой"** или **"Запустить"**
2. Дождись запуска (1-2 минуты)
3. Скопируй URL приложения (что-то вроде `https://zal-dance-api.timeweb.cloud`)

### 1.6. Проверь работу API
Открой в браузере: `https://твой-api.timeweb.cloud/api/health`
Должен показать: `{"status":"ok","timestamp":"..."}`

## Шаг 2: Загрузка фронтенда (сайт)

### 2.1. Обнови URL API в проекте
Перед сборкой нужно обновить файл `.env`:

Открой файл `/dance-space-zal2/.env` и измени:
```
VITE_API_BASE_URL=https://твой-api.timeweb.cloud
```
(замени на реальный URL твоего API из шага 1.5)

### 2.2. Собери проект
В терминале выполни:
```bash
cd /Users/matvejvoroncov/Downloads/lookbeta1/dance-space-zal2
npm run build
```
Готовые файлы появятся в папке `dist/`

### 2.3. Создай статический сайт в Timeweb
1. В левом меню нажми **"Статические сайты"** (или "Static Sites")
2. Нажми **"Создать сайт"**
3. Настройки:
   - **Название**: `zal-dance`
   - **Тариф**: Бесплатный

### 2.4. Загрузи файлы
1. Перейди во вкладку **"Файлы"**
2. Загрузи ВСЕ содержимое папки `dist/`:
   - `index.html`
   - `assets/` (папка)
   - все остальные файлы из dist
3. Или через FTP/SFTP если есть

### 2.5. Получи URL сайта
После загрузки получишь URL вроде:
`https://zal-dance.timeweb.cloud`

### 2.6. Обнови FRONTEND_URL в бэкенде
Вернись в настройки бэкенда (Apps → zal-dance-api → Переменные) и обнови:
```
FRONTEND_URL = https://zal-dance.timeweb.cloud
```
Перезапусти бэкенд!

## Шаг 3: Настройка домена (опционально)

Если у тебя есть домен (например, zaldance.ru):

### 3.1. Для фронтенда
1. Открой статический сайт
2. Перейди во вкладку **"Домены"**
3. Нажми **"Добавить домен"**
4. Введи свой домен: `zaldance.ru`
5. Следуй инструкциям по настройке DNS

### 3.2. Для API (поддомен)
1. Добавь поддомен `api.zaldance.ru`
2. Или используй URL от Timeweb

### 3.3. Обнови URL в настройках
После привязки домена обнови:
- В бэкенде: `FRONTEND_URL = https://zaldance.ru`
- В фронтенде: `.env` → `VITE_API_BASE_URL = https://api.zaldance.ru`
- Пересобери и перезалей фронтенд!

## Шаг 5: Проверка работы

### 5.1. Проверь сайт
1. Открой сайт в браузере
2. Проверь что фотографии загружаются
3. Проверь что календарь работает

### 5.2. Проверь бронирование
1. Выбери дату и время
2. Заполни форму
3. Нажми "Забронировать"
4. Должна открыться страница оплаты YooKassa

### 5.3. Тестовая оплата
Сделай тестовую оплату и проверь:
- ✅ Редирект обратно на сайт работает
- ✅ Статус бронирования обновился
- ✅ Email-уведомление пришло

## Частые проблемы

### Сайт не открывается
- Проверь что все файлы из `dist/` загружены
- Проверь что `index.html` есть в корне

### API не работает
- Проверь логи приложения в Timeweb
- Проверь что переменные окружения заданы
- Проверь URL API в `.env` фронтенда

### Оплата не работает
- Проверь что YOOKASSA_SHOP_ID и YOOKASSA_SECRET_KEY правильные
- Проверь что FRONTEND_URL содержит https://

## Обновление сайта

Когда нужно обновить сайт:
1. Внеси изменения в код
2. Выполни `npm run build`
3. Загрузи новые файлы из `dist/` в Timeweb (замени старые)
